\textbf{Core idea}

\begin{itemize}
  \item Python is a \y{layered ecosystem}
  \item Packages build on top of each other
  \item Higher layers assume lower layers exist
  \item This structure matters for:
    \begin{itemize}
      \item installation
      \item debugging
      \item performance
    \end{itemize}
\end{itemize}

\end{column}
